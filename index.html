AI Intern

I developed a voice-controlled desktop application called AI Intern. It can perform most computer tasks, such as opening files, running programs, and searching the web. AI Intern is powered by Google's Cloud Speech-to-Text API and Open AI API. It is trained on a dataset of millions of voice commands, so it can understand a wide range of commands and perform them accurately. AI Intern is still under development, but I am confident that it will be a valuable tool for users who want to operate their computers by voice commands.


Tech Stack- Python, OpenAI API, Google's Cloud Speech-to-Text API

Snake Game

I developed a desktop application called Snake Game using Python and Pygame. The game is very user-friendly and can be played by anyone. To play, simply move the snake around the screen using the arrow keys. The snake will grow longer as it eats apples. If the snake touches itself or the edge of the screen, the game is over. The game was developed as a personal project to learn more about Python and Pygame. I am happy with the results and I hope that others will enjoy playing the game.


Tech Stack- Python and Pygame.
